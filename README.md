# 🌱 身体のクセ診断アプリ

SEEK STUDIOの身体のクセをチェックする診断Webアプリケーションです。

## 🌐 デモ

（デプロイ後にURLを記載）

## ✨ 機能

- **3タイプ診断**: 呼吸・胸郭、骨盤・股関節、足部・足首
- **安全チェック**: レッドフラッグ（危険信号）の確認
- **詳細アドバイス**: 各タイプに応じた具体的なエクササイズ提案
- **バランス分析**: 視覚的なスコア表示
- **診断履歴保存**: 過去の診断結果を自動保存

## 🚀 ローカルで実行

```bash
pip install -r requirements.txt
streamlit run diagnosis_app.py
```

## 📱 技術スタック

- **Framework**: Streamlit
- **Language**: Python 3.9+
- **Data Visualization**: Pandas, Streamlit Charts
